Curso de JavaScript para Principiantes

Descripción del Curso

Este curso está diseñado para aquellos que desean aprender JavaScript desde cero. A lo largo de las lecciones, aprenderás cómo hacer que las páginas web sean interactivas utilizando este potente lenguaje de programación.

Duración del Curso

Duración total: 90 minutos

Objetivos del Curso

  • Comprender los conceptos básicos de JavaScript.
  • Aprender a manipular el DOM (Document Object Model).
  • Trabajar con variables, funciones y estructuras de control.
  • Desarrollar interactividad en una página web.

Contenido del Curso

  1. Módulo 1: Introducción a JavaScript (20 minutos)
  2. Módulo 2: Variables y Tipos de Datos (20 minutos)
  3. Módulo 3: Estructuras de Control (20 minutos)
  4. Módulo 4: Funciones y Eventos (20 minutos)
  5. Módulo 5: Proyecto Final (10 minutos)

Módulo 1: Introducción a JavaScript

JavaScript es un lenguaje de programación que se utiliza para agregar interactividad a las páginas web. Es uno de los lenguajes más importantes en el desarrollo web moderno.

Cómo Incluir JavaScript en una Página

  • <script>: Es la etiqueta que permite incluir código JavaScript en una página HTML.
  • src: Atributo que permite enlazar un archivo externo de JavaScript.

Módulo 2: Variables y Tipos de Datos

En este módulo, aprenderás a declarar variables y trabajar con los diferentes tipos de datos que JavaScript utiliza.

Declaración de Variables

  • var: La forma antigua de declarar variables (hoy menos utilizada).
  • let: Declaración de variables con un alcance local.
  • const: Declaración de constantes, cuyo valor no puede cambiar.

Tipos de Datos

  • string: Texto o cadenas de caracteres.
  • number: Números, ya sean enteros o decimales.
  • boolean: Valores verdadero (true) o falso (false).
  • array: Lista ordenada de valores.

Módulo 3: Estructuras de Control

Las estructuras de control nos permiten tomar decisiones y repetir acciones en nuestro código.

Estructuras de Condición

  • if: Permite ejecutar un bloque de código si se cumple una condición.
  • else: Se ejecuta cuando la condición no se cumple.
  • switch: Permite comparar una variable con diferentes opciones.

Bucles

  • for: Ejecuta un bloque de código un número determinado de veces.
  • while: Ejecuta un bloque de código mientras se cumpla una condición.

Módulo 4: Funciones y Eventos

Las funciones permiten agrupar código y los eventos nos permiten interactuar con los usuarios en la página web.

Funciones

  • Declaración de una función: Agrupar un bloque de código bajo un nombre.
  • Llamada de una función: Ejecutar el bloque de código cuando sea necesario.

Eventos

  • click: Se activa cuando el usuario hace clic en un elemento.
  • mouseover: Se activa cuando el usuario pasa el cursor sobre un elemento.

Módulo 5: Proyecto Final

En este módulo, crearás un proyecto práctico en el que utilizarás todo lo aprendido para crear una página interactiva con JavaScript.

  • Crear una página con un formulario interactivo.
  • Mostrar mensajes personalizados basados en las acciones del usuario.
  • Manipular elementos del DOM en tiempo real.

Conclusión

¡Felicidades! Has completado el curso de JavaScript para principiantes. Ahora puedes comenzar a crear páginas web más interactivas y dinámicas utilizando JavaScript.

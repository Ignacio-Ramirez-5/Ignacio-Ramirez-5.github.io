Curso de HTML y CSS

Descripción del Curso

Este curso está diseñado para principiantes que desean aprender los fundamentos del desarrollo web utilizando HTML y CSS. A lo largo de varias lecciones, exploraremos los conceptos básicos de la estructura de páginas web y cómo diseñarlas con estilo.

Duración del Curso

Duración total: 90 minutos

Objetivos del Curso

  • Aprender los fundamentos del lenguaje HTML.
  • Conocer cómo estructurar una página web.
  • Entender cómo dar estilo a las páginas web con CSS.
  • Desarrollar páginas web simples y funcionales.

Contenido del Curso

  1. Módulo 1: Introducción a HTML (20 minutos)
  2. Módulo 2: Estructura de una página HTML (20 minutos)
  3. Módulo 3: Introducción a CSS (20 minutos)
  4. Módulo 4: Estilizando tu página web (20 minutos)
  5. Módulo 5: Proyecto Final (10 minutos)

Módulo 1: Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. En este módulo, aprenderás cómo estructurar una página básica utilizando etiquetas HTML.

Etiquetas HTML Básicas

  • <html>: Define el documento HTML.
  • <head>: Contiene información sobre el documento, como el título.
  • <body>: Contiene el contenido visible de la página.

Módulo 2: Estructura de una Página HTML

En este módulo, profundizaremos en la estructura de una página web y aprenderemos cómo organizar el contenido de manera adecuada.

División de la Página

Una página HTML típica tiene una estructura básica que incluye elementos como encabezados, párrafos, listas y enlaces.

  • <h1> a <h6>: Encabezados para estructurar la jerarquía del contenido.
  • <p>: Define un párrafo de texto.
  • <a>: Crea enlaces hacia otras páginas o sitios web.

Módulo 3: Introducción a CSS

CSS (Cascading Style Sheets) es el lenguaje utilizado para estilizar y diseñar páginas web. En este módulo, aprenderás cómo aplicar estilos básicos a una página HTML utilizando CSS.

Selección de Elementos

  • Selector de tipo: Aplica estilos a todos los elementos de un tipo, como <p> o <h1>.
  • Selector de clase: Aplica estilos a elementos con una clase específica (ej. .mi-clase).
  • Selector de ID: Aplica estilos a un elemento con un ID único (ej. #mi-id).

Módulo 4: Estilizando tu Página Web

En este módulo, aprenderás cómo dar estilo a tu página utilizando diferentes propiedades CSS, como colores, fuentes, márgenes y más.

Propiedades de Estilo

  • color: Define el color del texto.
  • background-color: Define el color de fondo de un elemento.
  • font-family: Cambia la fuente de un texto.
  • margin: Controla el espacio exterior de un elemento.

Módulo 5: Proyecto Final

Al final del curso, realizarás un proyecto práctico donde aplicarás todo lo aprendido para crear una página web completa con HTML y CSS.

  • Crear una página de presentación personal utilizando HTML.
  • Aplicar estilos básicos con CSS, como fuentes, colores y márgenes.
  • Incorporar imágenes y enlaces a otras páginas o redes sociales.

Conclusión

¡Felicidades! Has completado el curso de HTML y CSS para principiantes. Ahora puedes comenzar a crear tus propias páginas web y aplicar estilos para hacerlas más atractivas.
